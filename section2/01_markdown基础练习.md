# Markdown 基础练习

- 通过动手练习巩固 Markdown 基础语法。
- 能在编辑器中实时预览并验证渲染效果。

## 前置条件

- 已阅读 [00_markdown基础语法](00_markdown基础语法.md)。
- 已安装 VS Code 或 Obsidian，可在本文件中直接编辑和预览。

以下练习由浅入深，请在 **编辑区** 中按提示内容填写 Markdown 语法，然后在预览中查看效果。

建议在 VS Code 中打开本文件，按 `Ctrl+K V` 打开侧边预览，一边编辑一边观察渲染变化。

## 参考答案提示

遇到困难时，可以参考以下提示：

- **标题**：`# 一级标题`
- **粗体**：`**文字**`
- **斜体**：`*文字*`
- **引用**：`> 引用内容`
- **无序列表**：`- 项目`
- **有序列表**：`1. 项目`
- **代码**：`` `代码` ``
- **链接**：`[显示文字](URL)`
- **图片**：`![替代文本](URL)`
- **分隔线**：`---`

> [!tip] 练习完成后
>
> 打开预览确认所有元素渲染正常。鼓励你将短文保存为独立笔记，作为你的第一篇 Markdown 作品！
