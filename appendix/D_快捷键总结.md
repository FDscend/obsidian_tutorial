# 常用快捷键总结

## VS Code 通用

| 快捷键                 | 功能                            |
| :--------------------- | :------------------------------ |
| `Ctrl+S`               | 保存文件                        |
| `Ctrl+Z`               | 撤销                            |
| `Ctrl+Y`               | 恢复（重做）                    |
| `Ctrl+X`               | 剪切                            |
| `Ctrl+C`               | 复制                            |
| `Ctrl+V`               | 粘贴                            |
| `Ctrl+F`               | 在当前文件中查找                |
| `Ctrl+H`               | 在当前文件中替换                |
| `Ctrl+Shift+F`         | 在全部文件中查找                |
| `Ctrl+P`               | 快速打开文件                    |
| `` Ctrl+` ``           | 打开/关闭终端                   |
| `Ctrl+B`               | 切换侧边栏                      |
| `Ctrl+Shift+P`         | 打开命令面板                    |
| `Ctrl+Shift+X`         | 打开扩展面板                    |
| `Ctrl+Shift+U`         | 打开输出面板                    |
| `Ctrl+K V`             | 打开侧边 Markdown 预览          |
| `Ctrl+Shift+V`         | 打开 Markdown 预览标签页        |
| `Ctrl+K Ctrl+T`        | 切换配色主题                    |
| `Alt+Z`                | 切换自动换行 / 不换行           |
| `Alt+Shift` + 鼠标点击 | 多选行（列编辑）                |
| `Ctrl+/`               | 注释/取消注释当前行或选中的代码 |
| `Ctrl+D`               | 选中下一个相同的词              |
| `Ctrl+Shift+L`         | 选中所有相同的词                |

## VS Code Git（源代码管理）

| 快捷键                             | 功能                           |
| :--------------------------------- | :----------------------------- |
| `Ctrl+Shift+G`                     | 打开源代码管理面板             |
| `Ctrl+Enter`                       | 提交更改（在提交信息输入框中） |
| `Ctrl+Shift+P` → `Git: 拉取`       | 从远程拉取                     |
| `Ctrl+Shift+P` → `Git: 推送`       | 推送到远程                     |
| `Ctrl+Shift+P` → `Git: 切换到分支` | 切换分支                       |

## Obsidian 通用

| 快捷键                         | 功能                      |
| :----------------------------- | :------------------------ |
| `Ctrl+P`                       | 打开命令面板              |
| `Ctrl+E`                       | 切换编辑/预览模式         |
| `Ctrl+N`                       | 新建笔记                  |
| `Ctrl+O`                       | 快速打开笔记              |
| `Ctrl+S`                       | 保存当前笔记              |
| `Ctrl+F`                       | 在当前笔记中搜索          |
| `Ctrl+Shift+F`                 | 在全部笔记中搜索          |
| `Ctrl+Click`（或 `Alt+Click`） | 打开 WikiLink 链接        |
| `[[`                           | 插入 WikiLink（自动补全） |

## Obsidian Canvas

| 快捷键                           | 功能                                                |
| :------------------------------- | :-------------------------------------------------- |
| `Ctrl` + 鼠标拖拽 / 鼠标中键拖拽 | 平移画布                                            |
| `Ctrl+滚轮` / 触控板双指缩放     | 缩放画布                                            |
| `Ctrl+0`                         | 适应画布大小                                        |
| `N`                              | 创建空文本卡片                                      |
| `Delete` / `Backspace`           | 删除选中的卡片或连接线                              |
| `Shift` + 拖拽卡片               | 保持水平/垂直对齐移动                               |
| `Ctrl` + 点击卡片                | 多选卡片                                            |
| `Ctrl+F`                         | 在当前 Canvas 中搜索文本（需 Advanced Canvas 插件） |
| `Ctrl+Z`                         | 撤销（Canvas 支持多步撤销）                         |
| `Esc`                            | 退出演示/翻页模式                                   |

## Obsidian Bases

| 快捷键                     | 功能          |
| :------------------------- | :------------ |
| `Ctrl` 或 `Alt` + 点击列头 | 按该列排序    |
| `Ctrl` + 点击已排序的列头  | 切换升序/降序 |

## GitHub Copilot

| 快捷键       | 功能                 |
| :----------- | :------------------- |
| `Ctrl+I`     | 打开内联对话         |
| `Ctrl+Alt+I` | 打开 Chat 面板       |
| `Tab`        | 接受当前代码补全建议 |
| `Ctrl+→`     | 逐词接受建议         |
| `Alt+[`      | 切换到上一个建议     |
| `Alt+]`      | 切换到下一个建议     |
| `Esc`        | 拒绝当前建议         |

## 常用 Markdown 语法速查

| 语法                    | 效果                       |
| :---------------------- | :------------------------- |
| `# 标题`                | 一级标题                   |
| `## 标题`               | 二级标题                   |
| `**粗体**`              | **粗体**                   |
| `*斜体*`                | _斜体_                     |
| `~~删除线~~`            | ~~删除线~~                 |
| `==高亮==`              | ==高亮==（Obsidian）       |
| `> 引用`                | 引用文本                   |
| `- 列表项`              | 无序列表                   |
| `1. 列表项`             | 有序列表                   |
| `[链接文字](url)`       | 超链接                     |
| `![替代文字](图片路径)` | 图片                       |
| `` `代码` ``            | 行内代码                   |
| ` `` `                  | 代码块                     |
| `- [ ] 任务`            | 未完成任务                 |
| `- [x] 任务`            | 已完成任务                 |
| `[[笔记名]]`            | WikiLink（Obsidian）       |
| `> [!tip] 提示`         | Callout 提示框（Obsidian） |
| `$E=mc^2$`              | 行内公式                   |
| `$$...$$`               | 块级公式                   |
| `---`                   | 分隔线                     |

## 常用终端命令

| 命令                       | 说明                              |
| :------------------------- | :-------------------------------- |
| `git clone <url>`          | 克隆远程仓库                      |
| `git status`               | 查看工作区状态                    |
| `git add <文件>`           | 暂存文件                          |
| `git commit -m "信息"`     | 提交更改                          |
| `git push`                 | 推送到远程                        |
| `git pull`                 | 从远程拉取                        |
| `git branch`               | 列出分支                          |
| `git checkout -b <分支名>` | 创建并切换分支                    |
| `git merge <分支名>`       | 合并分支                          |
| `cd <目录>`                | 切换目录                          |
| `ls` / `dir`               | 列出文件（Linux/macOS / Windows） |
| `mkdir <目录>`             | 创建目录                          |
