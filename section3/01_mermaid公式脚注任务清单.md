## 参考答案提示

遇到困难时，可以参考以下示例：

| 元素     | 语法                      |
| :------- | :------------------------ |
| 表格     | `\| 列1 \| 列2 \|`        |
| 脚注     | `正文[^1]` → `[^1]: 内容` |
| 删除线   | `~~文字~~`                |
| 任务列表 | `- [ ] 任务`              |
| Mermaid  | ` ```mermaid ... ``` `    |
| 行内公式 | `$E=mc^2$`                |
| 块级公式 | `$$...$$`                 |
| Callout  | `> [!note] 标题`          |
| WikiLink | `[[笔记名]]`              |

> [!tip] 渲染检查
>
> - Mermaid 不渲染时，检查代码块标签是否为 `mermaid`。
> - 数学公式显示为源码时，检查 `$` 是否成对出现。
> - Callout 显示为普通引用时，检查 `[!note]` 后是否空行。
